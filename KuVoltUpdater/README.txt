## KuVolt Updater.xml 편집 지침

Name에는 프로젝트 이름을 작성하세요.
Origin에는 원본 파일들이 위치한 주소를 작성하세요.
주소 마지막에는 반드시 슬래쉬(/)를 써주셔야 합니다.
예)
http://www.test.com/project/files = X
http://www.test.com/project/files/ = O

## 파일명 지침

KuVolt Updater는 파일명을 마음대로 바꾸어도 되지만 관련파일들의 이름을 모두 같도록 수정해야합니다.

### 관련 파일
 - KuVolt Updater.exe
 - KuVolt Updater.xml
 - 로고 이미지

## 로고 이미지 규격

사이즈 : 320*200
포맷 : png, jpg, gif, bmp

## 사용법
1. KuVoltProjectCreater.exe와 blacklist.txt파일을 게시할 프로젝트의 폴더로 복사하세요.
2. blacklist.txt에 제외할 파일들의 상대 경로를 작성하세요. 이때, 제외할 파일외에 다른 문자가 들어가서는 안됩니다.
3. KuVoltProjectCreater.exe를 실행하고 KuVolt.md5 파일이 생성될때까지 기다리세요.
4. 생성된 md5파일을 포함한 프로젝트 폴더의 파일들을 서버에 업로드하세요.

## 업로드시 주의사항
 - 주소창에 직접 파일의 위치를 입력하여 직접적인 파일 다운로드가 가능해야합니다.
 - 업로드시 파일을 압축해서 올리면 안됩니다.
